# Requirements: Space Invaders Futurista Mobile

**Defined:** 2026-03-23  
**Core Value:** Entregar uma experiência de combate espacial divertida, moderna e altamente rejogável no mobile, com controles simples e sensação forte de progresso a cada fase.

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### Core Gameplay

- [ ] **CORE-01**: Jogador consegue controlar nave por toque com latência baixa e resposta consistente.
- [ ] **CORE-02**: Jogador consegue pausar e retomar partida sem perder estado da run.
- [ ] **CORE-03**: Jogo permanece jogável offline para o loop principal.
- [ ] **CORE-04**: Jogo fornece feedback audiovisual claro em impactos, dano, derrotas e vitórias.

### Campanha e Progressão

- [ ] **CAMP-01**: Jogador consegue progredir por fases com objetivos e transições claras.
- [ ] **CAMP-02**: Cada fase possui chefe com mecânica distinta e telemetria de derrota/vitória.
- [ ] **CAMP-03**: Jogador consegue montar build de upgrades durante a run com sinergias identificáveis.
- [ ] **CAMP-04**: Jogo dispara eventos dinâmicos durante runs para variar ritmo e desafio.
- [ ] **CAMP-05**: Jogador desbloqueia progressão permanente entre partidas.

### Mobile Produto e Go-to-Market

- [ ] **MOB-01**: Aplicativo gera build distribuível para Android (Play Store).
- [ ] **MOB-02**: Aplicativo gera build distribuível para iOS (Apple Store).
- [ ] **MOB-03**: Jogo integra anúncios em pontos naturais sem interromper combate ativo.
- [ ] **MOB-04**: Jogo integra compras in-app para itens permitidos no modelo de monetização.
- [ ] **MOB-05**: Time executa soft launch controlado com métricas mínimas de estabilidade e retenção antes do lançamento amplo.

## v2 Requirements

Deferred to future release. Tracked but not in current roadmap.

### Comunidade e Expansão

- **COMM-01**: Jogador consegue competir em leaderboard global sincronizado em nuvem.
- **COMM-02**: Jogadores participam de narrativa estendida com arco de história.
- **COMM-03**: Jogadores participam de temporada com battle pass.

## Out of Scope

Explicitly excluded. Documented to prevent scope creep.

| Feature | Reason |
|---------|--------|
| Multiplayer online | Alto custo técnico/operacional para etapa atual; reduz foco no core loop |
| Clans/social completo | Alto escopo para v1 e baixo impacto no objetivo imediato de lançamento |
