# Requirements: Space Invaders Futurista Mobile

**Defined:** 2026-03-23
**Core Value:** Entregar uma experiencia de combate espacial divertida, moderna e altamente rejogavel no mobile, com controles simples e sensacao forte de progresso a cada fase.

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### Core Gameplay

- [ ] **CORE-01**: Jogador consegue controlar nave por toque com latencia baixa e resposta consistente.
- [ ] **CORE-02**: Jogador consegue pausar e retomar partida sem perder estado da run.
- [ ] **CORE-03**: Jogo permanece jogavel offline para o loop principal.
- [ ] **CORE-04**: Jogo fornece feedback audiovisual claro em impactos, dano, derrotas e vitorias.

### Campanha e Progressao

- [ ] **CAMP-01**: Jogador consegue progredir por fases com objetivos e transicoes claras.
- [ ] **CAMP-02**: Cada fase possui chefe com mecanica distinta e telemetria de derrota/vitoria.
- [ ] **CAMP-03**: Jogador consegue montar build de upgrades durante a run com sinergias identificaveis.
- [ ] **CAMP-04**: Jogo dispara eventos dinamicos durante runs para variar ritmo e desafio.
- [ ] **CAMP-05**: Jogador desbloqueia progressao permanente entre partidas.

### Mobile Produto e Go-to-Market

- [ ] **MOB-01**: Aplicativo gera build distribuivel para Android (Play Store).
- [ ] **MOB-02**: Aplicativo gera build distribuivel para iOS (Apple Store).
- [ ] **MOB-03**: Jogo integra anuncios em pontos naturais sem interromper combate ativo.
- [ ] **MOB-04**: Jogo integra compras in-app para itens permitidos no modelo de monetizacao.
- [ ] **MOB-05**: Time executa soft launch controlado com metricas minimas de estabilidade e retencao antes do lancamento amplo.

## v2 Requirements

Deferred to future release. Tracked but not in current roadmap.

### Comunidade e Expansao

- **COMM-01**: Jogador consegue competir em leaderboard global sincronizado em nuvem.
- **COMM-02**: Jogadores participam de narrativa estendida com arco de historia.
- **COMM-03**: Jogadores participam de temporada com battle pass.

## Out of Scope

Explicitly excluded. Documented to prevent scope creep.

| Feature | Reason |
|---------|--------|
| Multiplayer online | Alto custo tecnico/operacional para etapa atual; reduz foco no core loop |
| Clans/social completo | Alto escopo para v1 e baixo impacto no objetivo imediato de lancamento |
