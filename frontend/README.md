# Space Invaders — Modern UI (frontend)

Quick scaffold for development.

Install dependencies:

```bash
cd frontend
npm install
```

Run dev server:

```bash
npm run dev
```

Build:

```bash
npm run build
```

Notes:
- This scaffold provides a minimal Canvas-based renderer and HUD using Tailwind CSS.
- Configure Tailwind and Vite as needed; add assets under `src/assets`.
